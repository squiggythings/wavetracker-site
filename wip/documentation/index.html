Overview


This is the official documentation for WaveTracker. WaveTracker is a tracker for making chiptune music. There are up to 24 channels, up to 100 customizable waves, and up to 100 customizable instrument macros.


WaveTracker is an open source music software by Elias Ananiadis · Source code · Personal site
