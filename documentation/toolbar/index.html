Toolbar


The main toolbar, at the top of the window. The buttons are described from left to right:

Button Description
New Create a new song
Open Open an existing song
Save Save the current song to disk
Save as Save a copy of the current song in a new location
Export Render the song as a .wav
Undo Undo the last action
Redo Redo the last action
Cut Cut selected pattern
Copy Copy selected pattern
Paste Paste selected pattern
Play Play from the beginning of this frame
Play from beginning Play from the first frame
Stop Stop playback
Record / Edit mode Toggle edit mode on and off
Previous frame Go to the previous frame
Next frame Go to the next frame
Preferences Opens the preferences menu
Follow mode If this is enabled, the cursor will follow the playhead during playback
